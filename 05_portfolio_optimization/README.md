# Portfolio Optimization

Mean-variance optimization using Modern Portfolio Theory to construct optimal portfolios.

## Features

- **Efficient Frontier**: 10,000 Monte Carlo simulated portfolios
- **Optimal Portfolios**: Maximum Sharpe and Minimum Variance
- **Risk Metrics**: VaR, CVaR, Maximum Drawdown
- **Backtest**: 5-year historical performance comparison

## Asset Universe

- SPY (US Equities)
- EFA (International Developed)
- EEM (Emerging Markets)
- AGG (US Bonds)
- VNQ (REITs)
- GLD (Gold)
- TLT (Long Treasuries)
- USO (Oil)

## Results

- **Maximum Sharpe Portfolio**: 9.8% return, 10.2% volatility, 0.78 Sharpe
- **Backtest Performance**: 48.2% total return vs 72.4% SPY, but with 0.71 vs 0.55 Sharpe
