# Three Statement Financial Model

Integrated financial model connecting Income Statement, Balance Sheet, and Cash Flow Statement with DCF valuation.

## Overview

This project builds a fully integrated three-statement financial model from scratch using Python. The model follows the same logic used in investment banking and corporate finance.

## Features

- **Income Statement**: 5-year projections with revenue growth, cost structure, and profitability
- **Balance Sheet**: Working capital, PP&E, debt, and equity projections
- **Cash Flow Statement**: Operating, investing, and financing activities
- **DCF Valuation**: Enterprise value from discounted free cash flows
- **Sensitivity Analysis**: WACC vs terminal growth rate matrix

## Key Assumptions

| Assumption | Value |
|------------|-------|
| Revenue Growth | 8% |
| Gross Margin | 42% |
| WACC | 10% |
| Tax Rate | 25% |
| Terminal Growth | 2.5% |

## Results

- **Enterprise Value**: $1.96B
- **Equity Value**: $1.91B
- **Implied Share Price**: $19.11
