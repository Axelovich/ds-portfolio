Project Overview

This project builds a fully integrated three-statement financial model from scratch using Python. The model connects the Income Statement, Balance Sheet, and Cash Flow Statement following the same logic used in investment banking and corporate finance.

The model includes a DCF (Discounted Cash Flow) valuation module that calculates enterprise value and implied share price based on projected free cash flows.

Key Takeaways

  • The model is fully integrated: changes in assumptions automatically flow through all three statements
  • Balance sheet balances in all periods (Assets = Liabilities + Equity)
  • Cash flow reconciles: ending cash matches balance sheet cash position
  • DCF implies 96% upside from a hypothetical $10 current share price
  • Valuation is most sensitive to terminal growth rate at lower WACC levels
